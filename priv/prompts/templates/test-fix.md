# Test Fix

## Contract
You will produce: passing tests for the specified failures.
You will NOT: change production code behavior or skip/disable tests.
The output enables: green CI without masking real failures.

## Validation
Run `pnpm exec vitest run` and ensure all tests pass before committing.

## Scope
Fix the failing tests. If the test is correct and the implementation is wrong, fix the implementation, but only the minimal change needed.
